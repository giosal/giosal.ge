SFI test task v.0.1
Author: George Salukvadze
Date: 2019-05-05

The webpage represents the test challenge given by Swiss Finance Institute within an interview process for developer position.

The following technical stack was used:
- HTML to create a page
- JavaScript and jQuery for main functionality
- PHP to read Excel file + SimpleXLSX class
- HighCharts to display the chart + DraggablePoints plugin

The webpage runs as is and doesn't require any additional setup.
